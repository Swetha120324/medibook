# 🏥 MediBook

MediBook is a smart healthcare appointment booking web application.

## 🚀 Features
- Book doctor appointments
- Symptom-based doctor suggestion
- Time slot selection
- Download booking receipt
- Admin dashboard to manage appointments
- Emergency hospital finder (Google Maps)

## 🛠️ Technologies Used
- React.js
- Vite
- CSS
- LocalStorage
